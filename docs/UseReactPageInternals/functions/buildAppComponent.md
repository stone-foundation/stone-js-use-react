# Function: buildAppComponent()

```ts
function buildAppComponent(
   event, 
   container, 
   component?, 
   layout?, 
   data?, 
   statusCode?, 
error?): Promise<ReactNode>;
```

Or for the main handler if the route is not defined.

### statusCode?

`number`

### error?

`any`
